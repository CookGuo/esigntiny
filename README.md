# esigntiny

利用 `tingpng` 的压缩小工具

## Usage
### Nodejs
```javascript
var esigntiny = require('esigntiny')

esigntiny({
  input: './images',
  output: './images',
})
```
### Command Line
```javascript
npm install esigntiny -g

esigntiny start -i input[default /assets/images] -o output[default /assets/images]
```
